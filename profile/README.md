## 🦙 Featured: LAMA — Live Auction Market Assessor

Real-time price checking overlay for **Path of Exile 2**. Copy any item → instant market data from poe.ninja.

- **`Ctrl+C`** — Instant price overlay near your cursor
- **`Ctrl+Shift+C`** — Deep trade API query with actual mods, DPS, and defense
- **`Ctrl+Shift+B`** — Built-in bug reporter

Free forever. No ads. No tracking. GPL-3.0.

**→ [Get LAMA](https://github.com/CouloirGG/lama)**
